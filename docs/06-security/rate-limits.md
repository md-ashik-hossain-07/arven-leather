# Rate Limiting

Version: 2.0

Status: Completed

---

# Purpose

Protect APIs against abuse and denial-of-service attacks.

---

# Limits

Anonymous Users

100 Requests / Minute

Authenticated Users

300 Requests / Minute

API Clients

Configurable

---

# Protection

- IP Throttling
- Token Throttling
- Retry Delay
- Temporary Blocking

---

# Monitoring

Track:

- Request Count
- Blocked Requests
- Abuse Attempts

---

# Success Criteria

The API should remain available under normal traffic while resisting abuse.
